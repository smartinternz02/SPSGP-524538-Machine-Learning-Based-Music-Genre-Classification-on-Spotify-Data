About the Project

The Machine Learning Based Music Genre Classification on Spotify Data project aims to predict music genres using machine learning algorithms trained on Spotify data. By leveraging the vast amount of information available in the audio features of songs, the project seeks to automatically classify songs into different genres based on their characteristics.

The project utilizes a dataset consisting of a large collection of songs from various genres available on Spotify. These songs are associated with audio features such as tempo, energy, danceability, instrumentalness, loudness, speechiness, and valence. Machine learning algorithms, such as decision trees, random forests, or neural networks, are trained on this dataset to learn the patterns and relationships between the audio features and the corresponding genre labels.

By leveraging the trained machine learning models, users can input the audio features of a song, and the system will predict its most likely genre. This project not only provides a convenient tool for music enthusiasts and industry professionals but also showcases the potential of machine learning in music analysis and classification.


Result

After training and evaluating the machine learning models on the Spotify dataset, the project achieved an accuracy of 78% using the Bagging Classifier algorithm for music genre classification. This accuracy score indicates the percentage of correctly predicted genres for the given songs.
